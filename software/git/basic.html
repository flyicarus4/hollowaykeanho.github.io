About GIT

Software Configuration Management (SCM)

In Software development, we track development progress using GIT from time to time with available software like Github and Bitbucket.

To maximize to full potentials from SCM, each and every members needs to have a common mindset in term of processes. There are many SCM methods across the globe and companies. You'll need to visit the SCM person to get to know more about the processes being used for the team.

This guide is created to introduce a simple SCM using GIT on both Github and Bitbucket.

Using GIT

GIT is a widely used open-source software for managing software development due to its simplified features and distributed management.

Before we start, click on the links above and get to know more about GIT. This guide will cover the 'how to use' parts. Please note that to learn git effectively, you should do hands-on effort with a partner.

Also, it is advised to exercise "Don't know, don't try! Ask." while getting stucked when using GIT.
